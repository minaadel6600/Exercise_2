

# prerequisites : 
    1. nodejs version 16.14.0 or heigher 

# to run the task : 
    1. clone this repository "https://github.com/minaadel6600/Exercise_2.git"
    2. run command (npm install)
    3. put your data into file (input_file_name.csv) in the root folder for now it contains sample data which exist
        in the task description 
    4. then run command (node index.js)

    5. the output data files be generated in the root folder with names ( 0_order_log00.csv , 1_order_log00.csv)
        * Note :  0_order_log00.csv contains average quantity for every product 
        * Note :  1_order_log00.csv contains most popular Brand for every product
